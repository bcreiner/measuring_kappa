Measuring Kappa
===================

Simulation code used within the following paper:

Reiner, R.C. Jr, Guerra, C., Donnelly, M.J., Bousema, T., Drakeley, C., Smith, D.L (Sumbitted) **Measuring malaria transmission from humans to mosquitoes**.
